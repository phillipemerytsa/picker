Random Drug Test Room Selector

Directions

  1. Leave the checkbox checked for occupied rooms.
  2. Uncheck rooms that should NOT be included.
  3. Enter the percentage of occupied rooms to randomly select.
  4. Optionally enter room numbers to exclude manually (one per line).
  5. Click Run Random Selection.
  6. Use the print button if you want a paper copy.
